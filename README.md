# Personal Portfolio – React + Vite + Tailwind + shadcn-ui

This repository contains the source code for my personal portfolio website, built using modern frontend technologies.  
It showcases my VR development work, game development projects, web development skills, certifications, and research work.

---

## 🚀 Project Info

This is a fully responsive, interactive portfolio built with:

- **React (TypeScript)**
- **Vite**
- **Tailwind CSS**
- **shadcn/ui component library**

The website includes:

- Hero section with typing animation
- About me section (fresher-friendly)
- Skills section with icons
- Projects with demo videos + Tech Used badges
- Certifications grid with modal zoom
- Contact section with a working form + social links
- Clean neon UI with smooth animations

---

## 🛠️ How to Run the Project Locally

### **Prerequisites**

You must have **Node.js** and **npm** installed.  
(Recommended: Install with `nvm`)

### **Steps**

```sh
# Clone the repository
git clone <YOUR_REPOSITORY_URL>

# Go into the project directory
cd <YOUR_PROJECT_NAME>

# Install dependencies
npm install

# Start development server
npm run dev
```
