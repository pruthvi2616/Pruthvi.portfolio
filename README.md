# 🌐 Personal Portfolio Website

🔗 **Live Website:** https://pruthvi-portfolio-18.vercel.app  

This is my personal portfolio website showcasing my skills, projects, and experience in **Game Development, Virtual Reality (VR), and Software Development**. The portfolio highlights real-world projects, research work, and hands-on technical expertise.

---

## ✨ Features
- Modern, clean, and responsive UI  
- Project showcase with demo videos  
- Skills displayed with technology icons  
- Certificates and achievements section  
- Smooth animations and interactive design  
- Optimized for desktop and mobile devices  

---

## 🛠️ Tech Stack
- **Frontend:** React, TypeScript  
- **Styling:** Tailwind CSS  
- **UI & Animations:** Framer Motion  
- **Icons:** Lucide / FontAwesome  
- **Deployment:** Vercel  

---

## 📂 Sections Included
- Home  
- About Me  
- Skills  
- Projects (with video previews)  
- Certificates  
- Contact
